# 📊 Sales Data Analysis Dashboard (Power BI)

## 📌 Project Overview

This project is an interactive **Power BI Sales Dashboard** created to analyze business sales performance using a sales dataset. The dashboard provides insights into revenue, products, customers, regions, and sales trends to help businesses make data-driven decisions.

---

## 🎯 Objectives

- Analyze overall sales performance.
- Identify top-selling products.
- Compare sales across countries and territories.
- Understand monthly and yearly sales trends.
- Analyze order status and deal size distribution.
- Create an interactive dashboard for business decision-making.

## 📌 Key Insights

- Classic Cars generated the highest sales.
- Large Deal Size contributed the maximum revenue.
- Sales increased significantly during the last quarter.
- Most orders were successfully shipped.
- Certain countries contributed more revenue than others.

## 🚀 How to Use

1. Download the repository.
2. Open **sales data report.pbix** using **Power BI Desktop**.
3. Refresh the dataset if required.
4. Explore the dashboard using slicers and filters.
